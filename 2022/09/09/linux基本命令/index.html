linux系统方面

uname 查看操作系统

uname -a 查看详细信息

shutdown 无参数默认1分钟后关机

shutdown -c 取消关机

shutdown now 立刻关机

shutdown -r 重启

poweroff 立刻关机

reboot 重启

dpkg-reconfigure locales 更改语言，需要root权限(Debian)

service iptables start/stop 防火墙开/关，需要root权限

chkconfig iptables on/off 防火墙永久开/关，需要root权限

passwd 更改密码，需要root权限

linux用户方面

在命令行的末尾有#或者$符号

# = root用户

$ = 其他用户

root用户的家目录 /root

其他用户的家目录 /home/用户名

/etc/passwd 该目录存放用户信息

/etc/shadow 该目录存放用户密码信息(加密)

用户管理相关

提权相关

vi /etc/sudoers

需要root用户编辑该命令，以给对应用户定义具体权限

linux文件方面

文件管理相关

注：rm -rf /* 删除根目录下所有文件(危险命令，自己虚拟机爽爽差不多得了，在别人电脑或者企业电脑上乱搞容易进去)

讲一下文件权限

drwxr-xr-x 3 root root 4096 9月 21 21:53 img

这是随便复制的一条文件详细信息

这里写的分开一点 d rwx r-x r-x

第一个位置是这个文件是什么 d 文件夹 - 文件 l 链接

第二个位置是所有者权限 r=read=4 4代表可读 w=write=2 2代表可写 x=execute=1 代表可执行 这里的421表示权限数字，文件有什么权限就是数字加起来，比如7就是可读可写可执行

第三个位置是所有者所在组权限

第四个位置是其他人权限

文件编辑方面

这里介绍一下vi，vi编辑器是所有Unix和Linux系统下标准的编辑器，vim相当于vi的加强版，有程序编辑的能力，可以以字体颜色判断语法的正确性等。

vi和vim命令基本相同，这里只介绍vi的命令

vi [文件名] 编辑文件

进入文件页面后

按下i进入insert模式，然后可以开始编辑了

结束编辑后按下Esc退出编辑模式，进入命令模式

按下:进入末行模式

然后输入wq保存并退出(w保存/写入 q退出)

如果输入q!强制退出(不保存退出)

在命令模式下(就是刚进文件界面和按Esc退出编辑模式后的那个模式)

按下0将光标移到行首

按下$将光标移到行尾

按下:[数字]跳转行数

按下:set nu显示行号

按下set nonu取消显示行号

但是有一说一，如果你嫌麻烦，用键盘上下左右键移动也一样。。。

vi内部的复制

1.Esc退出编辑模式

2.按下v，通过移动光标来选择要复制的内容

3.按下y复制，会自动进入命令模式

4.调整光标，按下p粘贴

但有一说一如果我个人认为，这个非常麻烦还要额外记，不如直接

1.鼠标选中

2.ctrl+shift+c复制

3.ctrl+shift+v粘贴

(除非你是用那种只能用键盘控制，连鼠标都用不了，只有单纯一个命令行的linux系统)

vi的搜索

命令模式下\[搜索的内容]进行搜索

按下n搜索下一个
